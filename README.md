## 💻 Como usar

- **_Clone o projeto:_** <br>
  ` git clone https://github.com/lmiguelm/Gameplay`

- **_Entre na pasta do projeto:_** <br>
  `cd Gameplay`

- **_Baixe as dependências:_** <br>
  ` yarn || npm install`

- **_Rodar o app:_** <br>
  `expo start`
